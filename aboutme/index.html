Swarm Robotics Research

Polytechnique Montréal

In the Summer of 2018, I was working in the MIST Lab in Montréal researching decentralized algorithms for the cooperative transport of fabrics using micro-UAVs. The lab uses a novel programming language called Buzz to implement decentralized algorithms for swarm robotics. I integrated Buzz (which is compiled with a C compiler) in Python to interface with the Crazyflies via the Crazyradio dongle. I summarized my results in a paper that I presented at the 2019 International Conference on Robotics and Automation (ICRA).

Experimental Physics Research

University of Würzburg, Germany

From May until September of 2017, I worked alongside graduate students performing physics research at the University of Würzburg. We were investigating the electronic structure of spin-orbit–coupled surfaces with atomic spatial resolution. We used a Scanning Tunnelling Microscope (STM) to image the atoms on the surfaces of the substrates, and it was my responsibility to operate the STM and to decipher how the spins were oriented on the surface of the samples. I suggested the idea that MnO chains have a chiral magnetic structure, which is the main finding in our team's publication in Nature Communications.

Topology Tool Kit (TTK)

Topological Data Analysis Clustering Algorithm

Given a set of points, it is extremely useful for applications like machine learning to have a computer partition the points into clusters of high density. A classmate and I used a concept from topological data analysis called persistence to categorize points into clusters. The algorithm automatically decides how to assign these clusters with no user input, but also allows the user to manually change parameters. We have implemented our algorithm as an extension of TTK, and we worked directly with the creators of TTK to achieve this.

Increment Fitness

Eng Phys Capstone Project

Five of my classmates and I were granted $10,000 to found a business that would transform exercise tracking. Our team's vision was to create a product that could be installed at the gym to track the gym users' exercises, and feed that information to their smartphones. To prove that this was a viable concept, we created a prototype that could recognize 10 different exercises using machine learning and computer vision. We were the only project in Engineering Physics to win the Industry Award, granted by UBC Applied Science. Here is an excerpt from our final reccomendation report:

"Once connected to a computer with the corresponding software, the user can begin recording with the Xbox Kinect. The user must then input their unique user id into the software. The user can then stand about two meters away from the camera, and begin any one of the 10 allowable exercises with any of the 3 allowable weights at any time. Shortly after the user stops exercising, the system will send to an online database the exercise they did, along with information about the weight and number of repetitions. The user can walk in and out of frame as they please, and can begin any other exercise thereafter."

Swarm

A Video Game I Created

During one of my reading breaks, I set out to make my first video game. The game is written in Python, and the GUI was created with pygame. I started writing Swarm with hopes of making it playable on a Super Nintendo Entertainment System (SNES), so I made the game such that can be played using only 7 buttons. The game was "finished" well before I modified it for the SNES, but to do so, I had to get around the SNES controller timing protocol. Every 20 ms, the SNES sends the controller a portion of a 83 kHz square wave, and on a different line, the controller sends back a high pulse during the first period of the square wave if "B" is pressed, the second period if "Y" is pressed, and so on for each button on the controller. My Raspberry Pi could not read the GPIO pins fast enough to correctly distinguish which buttons were being pushed, so I had to program an FPGA to convert this timing protocol into 7 wires running to my Pi, where each wire was logic high if its corresponding button was pressed, and low otherwise. It worked like a charm!

When I finished the SNES project, I still continued to update the PC game. I have spent a lot of time playing it, so much so in fact, that my high score is Level 21. You can download the latest version below. I have recently updated the game such that it can be played over one's home network with another PC. The source code can be found here.
