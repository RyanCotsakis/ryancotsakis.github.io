Increment Fitness

Eng Phys Capstone Project

Five of my classmates and I were granted $10,000 to help start a business that would transform exercise tracking. For our second project course in Engineering Physics, we developed a prototype of a technology that with further development can be installed at any gym that tracks gym users' exercises with minimal effort from the user. The idea is that one would be able to download an app that we are creating, go to the gym, workout, and have all of their workout information logged on their phone without any manual entry of information. We were the only project in Engineering Physics to win the Industry Award, granted by UBC Applied Science. Here is an excerpt from our final reccomendation report:

"The system that this document describes is a proof of the concept that it is possible to track one’s exercise using machine learning algorithms. With that being said, the operation of this system is to be carried out in a laboratory setting. A Microsoft Kinect is to be set up one meter above ground, facing horizontally. Once connected to a computer with the corresponding software, the user can begin recording with the Kinect. The user must then input their unique user id into the software. The user can then stand about two meters away from the camera, and begin any one of the 10 allowable exercises with any of the 3 allowable weights at any time. Shortly after the user stops exercising, the system will send to an online database the exercise they did, along with information about the weight and number of repetitions. The user can walk in and out of frame as they please, and can begin any other exercise thereafter."

Swarm

A Video Game I Created

During my reading break, I set out to make my first video game. The game is written in Python, and the GUI was created with pygame. I started writing Swarm, with hopes of making it playable on a Super Nintendo Entertainment System (SNES); so I made the game such that can be played using only 7 buttons. The game was "finished" well before I modified it for the SNES, but to do so, I had to get around the SNES controller timing protocol. Every 20 ms, the SNES sends the controller a portion of a 83 kHz square wave, and on a different line, the controller sends back a high pulse during the first period of the square wave if "B" is pressed, the second period if "Y" is pressed, and so on for each button on the controller. My Raspberry Pi could not read the GPIO pins fast enough to correctly distinguish which buttons were being pushed, so I had to program an FPGA to convert this timing protocol into 7 wires running to my Pi, where each wire was logic high if its corresponding button was pressed, and low otherwise. It worked like a charm!

When I finished the SNES project, I still continued to update the PC game. I have spent a lot of time playing it, so much so in fact, that my high score is Level 21. You can download the latest version below. I have recently updated the game such that it can be played over one's home network with another PC (This feature is in its early stages; it may be buggy).
